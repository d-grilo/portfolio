Project Title

This project was a part of my formation. It is an intranet where all members of the company are registered. There are three main features:

- A Wiki that hosts categories and articles.

- A Project manager where users can manage their projects and tasks associated with those projects

- A Calendar that displays all relevant events, such as birthdays, vacations and project/task start and end dates. It is also possible to click on a particular day of the calendar to create an event for that day. The calendar also has an annual view where all user vacations are displayed.
